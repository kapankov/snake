# The Snake game for the terminal

This repository contains several different implementations of the Snake game for terminals, written in Pascal, C, and C++.

pas/snake.pas - cross-platform implementation in Free Pascal

src/snake.c - cross-platform implementation in C99

src/snake.cpp - cross-platform implementation in C++11

winsnake/snake.cpp - implementation in C++ for Windows (Classic Windows Console)

Cross-platform implementations work with escape sequences and are supported in Windows Terminal starting from Windows 10 (build 1607+), as well as in any Linux, Unix and MacOS.
